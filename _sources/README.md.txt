pyman
=====

Introduction to Python for Science -- local UBC fork

This is aimed at undergradate science students who want to learn Python for scientific applications.  No programming experience is assumed.  Suggestions for improvements are welcome.  [Create an issue](https://github.com/a301-teaching/pyman/issues) and let me know.
